# Linux Production Troubleshooting

This repository documents real-world Linux production issues and troubleshooting procedures encountered while working in enterprise and managed service environments.

It serves as a personal knowledge base and operational runbook for diagnosing and resolving infrastructure problems.

---

## Purpose

This repository is used to:

- Document production troubleshooting scenarios
- Capture operational knowledge
- Maintain reusable troubleshooting runbooks
- Share Linux systems administration practices
